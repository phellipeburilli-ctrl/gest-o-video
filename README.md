# Dashboard de Performance Audiovisual

Professional analytics dashboard for video editors, powered by Next.js, ClickUp API, and Shadcn/UI.

## Features
- **Dark Mode only** (Enterprise aesthetics)
- **Real-time ClickUp Data**
- **Performance KPIs**: Volume, Efficiency, Agility
- **Visualizations**: Recharts (Bar, Pie)

## Tech Stack
- Next.js 14+ (App Router)
- Typescript
- Tailwind CSS
- Shadcn/UI
- Lucide React

## Setup
1. Clone repo.
2. `npm install`
3. Configure `.env.local` with `CLICKUP_API_KEY` and `CLICKUP_LIST_ID`.
4. `npm run dev`
